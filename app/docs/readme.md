
#### Header头

**公共参数说明**

| 参数名         | 类型     | 说明     |
|:------------|:-------|:-------|
| X-Pub-Key   | string | 用户的公钥  |
| X-UID       | string | 用户的地址  |
| X-Sign      | string | 签名     |
| X-Time      | int64  | 时间戳    |
| X-Data-Hash | string | 数据hash |
